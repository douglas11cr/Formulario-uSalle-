# Sistema de Gestión para una Comercializadora de Productos

Este proyecto corresponde a la tercera fase del proyecto final de Interacción Humano Computador.

## Descripción

Prototipo web para una empresa comercializadora de productos, enfocado en la gestión de inventario, productos, ventas y reportes básicos.

## Funcionalidades del prototipo

- Inicio de sesión simulado.
- Panel principal con indicadores.
- Tabla de productos.
- Visualización de inventario.
- Alerta de bajo stock.
- Registro simulado de ventas.
- Reportes básicos.

## Tecnologías utilizadas

- HTML
- CSS
- JavaScript

## Cómo publicar en GitHub Pages

1. Crear un repositorio en GitHub.
2. Subir los archivos `index.html`, `estilos.css` y `script.js`.
3. Entrar a Settings.
4. Ir a Pages.
5. En Source seleccionar `Deploy from a branch`.
6. En Branch seleccionar `main` y carpeta `/root`.
7. Guardar.
8. Esperar a que GitHub genere el enlace público del sitio.
